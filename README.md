# 🏥 Hospital Management System

## 📌 Overview

Hospital Management System is a console-based application developed in C++ to simplify hospital operations by managing patient records, doctor information, appointments, and billing. The project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts and data structures to simulate real-world hospital workflows.

## 🎯 Objectives

- Manage patient and doctor records efficiently.
- Implement dynamic data storage using linked lists.
- Simulate hospital workflows such as doctor assignment and billing.
- Demonstrate the use of Object-Oriented Programming principles.


## 🚀 Features

- Patient Registration
- Doctor Management
- Patient Search
- Doctor Assignment
- Billing System
- Dynamic Memory Allocation
- Console-Based Menu Interface


## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project in any C++ IDE (VS Code, CodeBlocks, Dev-C++, etc.).
3. Compile the source file.

```bash
g++ hospital_management_system.cpp -o hospital
```

4. Run the executable.

```bash
./hospital
```
